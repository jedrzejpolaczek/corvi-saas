# Corvi – Instrukcja użytkownika

1. **Wgraj dane** (CSV/Parquet) → zakładka *Datasets*.
2. **Wybierz lub wgraj model** (sklearn/pkl/DL) → zakładka *Models*.
3. **Ustaw parametry**: metryka, budżet (liczba prób / czas), walidacja.
4. **Uruchom HPO**: *Experiments* → wybierz algorytm (Grid/Random; **Corvi‑Opt** w planach Premium) i backend (Local/Ray).
5. **Podgląd na żywo**: status, tabele prób, najlepszy wynik, krzywe.
6. **KPI/ROI**: zdefiniuj formułę, zapisz w projekcie.
7. **Eksporty**: CSV/Parquet i PDF (przycisk *Pobierz raport*).

## Limity Freemium
- Do 10 eksperymentów / miesiąc.
- Tylko Grid/Random.
- Brak KPI/ROI i BI/Exports.
